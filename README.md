# Microsystem-Linux
Microsystem Linux is an Puppy/Slackware based distro which runs on old computers.

## Data files
The files on the repo are generated files from the Puppy Remaster tool. This could be important or not.

### Distro and ISO
The ISO in the release section is an multi-image ISO file which means it could be used with UEFI or MBR.
The Distro itself has some tweeks from S15-Pup32. UPDATES are coming.
The installing progress is identical to S15-Pup32.
